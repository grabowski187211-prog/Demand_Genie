Source of truth: https://openai.devpost.com/

## Required before submission

- [ ] Confirm the entrant type.
- [ ] Confirm country of residence and eligibility.
- [ ] Select exactly one category:
  - Apps for Your Life
  - Work & Productivity
  - Developer Tools
  - Education
- [ ] Finish a working project that runs consistently on its target platform.
- [ ] Add complete setup and run instructions to `README.md`.
- [ ] Add sample data or test credentials if the app needs them.
- [ ] Document how Codex accelerated the workflow.
- [ ] Document where key product, engineering, and design decisions were made.
- [ ] Document how GPT-5.6 and Codex contributed to the final result.
- [ ] Record the `/feedback` Codex Session ID.
- [ ] Create a public YouTube demo video under 3 minutes.
- [ ] Make sure the video includes audio explaining the project and Codex/GPT-5.6 usage.
- [ ] Provide the repository URL in the submission.
- [ ] If the repository is private, share it with `testing@devpost.com` and `build-week-event@openai.com`.
- [ ] Provide a working demo URL or judging instructions if available.
- [ ] Submit on Devpost before July 21, 2026 at 5:00 PM PT.
